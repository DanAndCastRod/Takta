Diagnosticar y corregir un problema en Takta.

1. Pedir al usuario descripcion del sintoma y donde ocurre
2. Leer el archivo relevante con Read
3. Usar Grep para buscar el sintoma en el codebase
4. Leer backend/app/main.py si es un error de routing
5. Formular hipotesis explicita antes de editar
6. Editar con Edit (no reescribir archivos completos)
7. Describir el cambio realizado y su razon
8. Actualizar walkthrough.md con el diagnostico y fix
